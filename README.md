# UrbanGo API 

UrbanGo is a full-stack travel app that helps tourists find nearby food spots and attractions when visiting new cities.  
This repository contains the **Spring Boot backend** that powers the UrbanGo web app.

---

## What It Does
- Provides REST APIs for city and place search  
- Connects to external sources like Foursquare and OpenTripMap  
- Stores city and place information in a SQL database  
- Sends data to the React frontend for display on a map

---

## Tech Stack
- **Backend:** Spring Boot (Java)
- **Database:** PostgreSQL / SQL
- **External APIs:** Foursquare, OpenTripMap, OpenCage, OpenRouteService
- **Build Tool:** Maven

---

##  How To Run 
1. Make sure **Java 21** and **PostgreSQL** are installed.  
2. Clone this repository:  
   git clone https://github.com/yourusername/UrbanGo-API.git
3. Open it in your IDE and run the project (mvn spring-boot:run).
4. The API should start at http://localhost:8080
